# Saturn을 소개할게요!
Saturn은 간단한 다기능 디스코드 애플리케이션이에요. 이 프로젝트는 디스코드 서버를 운영하면서 수많은 애플리케이션들을 한 서버에서 복잡하게 사용해야 하는 불편함을 해소하기 위해 만들어졌어요.
유용한 기능들을 모아 완벽에 가까운 올인원 애플리케이션 완성을 목표로 하고 있어요.

Saturn은 현재로써 서버 관리, 음악 재생, 티켓 시스템 등 여러 가지 기능을 제공하고 있어요.

프로젝트에 대한 모든 형태의 기여, 제안, 버그 보고 또는 기능 요청을 환영해요.

# 기능
- 서버 관리 (멤버 차단, 추방, 메세지 삭제, 채널 생성 등)
- 음악 재생
- 티켓 시스템
- 멤버 인증 시스템
- 투표 시스템
- 보안 시스템 (도배 감지, 자동 차단 등)
